PRUEBA DE CONCEPTO — PORTAL DE OPORTUNIDADES DE FINANCIAMIENTO

1. Abra index.html en cualquier navegador moderno.
2. El prototipo funciona sin servidor y sin conexión a Internet, excepto los enlaces externos.
3. Incluye 12 oportunidades representativas extraídas del documento Word suministrado.
4. La fecha de referencia usada para clasificar oportunidades es 15 de julio de 2026.
5. Es una demostración no oficial inspirada en la identidad visual general de UNPHU.
6. Para publicar: suba la carpeta a Netlify, GitHub Pages, Vercel o Cloudflare Pages.
